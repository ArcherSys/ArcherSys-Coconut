Display custom Google Maps with WordPress widget. Open Google Map in popup window.

== Description ==

Simple [Google Maps plugin](http://readyshoppingcart.com/product/google-maps-plugin/ "Google Maps plugin") allow to great widget with Google Map or image. And open map in popup when you click on the image.

== Installation ==

1. Download Google Map Widget plugin
2. Upload to your WordPress plugins directory
3. Activate and enjoy

For more information review google maps widget plugin [FAQ section](http://readyshoppingcart.com/product/google-maps-widget-plugin/ "Google Maps widget")
